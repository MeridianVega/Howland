## Features

- **AI Code Conversion**: Specialized models for VB6, legacy C#, and more

- **Model Tiers**: Electra (fast), Noonan (balanced), Vega (premium)

## Installation

### Windows

1. Download `Howland-Setup.exe`
2. Run the installer
3. Follow the setup wizard
4. Howland will launch automatically

**Portable Version**: Download `Howland-Portable.exe` and run directly (no installation).

### macOS

1. Download `Howland.dmg`
2. Open the DMG file
3. Drag Howland to Applications folder
4. Launch from Applications

**First launch**: Right-click → Open (to bypass Gatekeeper)

### Linux

**AppImage**:
```bash
chmod +x Howland.AppImage
./Howland.AppImage
```

**Debian/Ubuntu**:
```bash
sudo dpkg -i howland.deb
sudo apt-get install -f
howland
```

## Usage

### Desktop Shortcuts

- **Alt+H** - Open assistant popup (quick access)
- **Alt+Shift+H** - Show main window

### Tray Icon

The app runs in your system tray:
- Left-click: Show assistant
- Right-click: Menu (settings, exit)

---

## Auto-Updates

Howland automatically checks for updates and downloads them in the background. Updates are installed when you restart the app.

To manually check: Right-click tray icon → Settings → Check for Updates
